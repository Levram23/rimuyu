# rimuyu
🎨 Concepto de Marca
Nombre: Rimuyo (del Quechua "hablar juntos")
Colores: Azul profundo del lago (#0A2647), Dorado andino (#C9A227), Azul cielo (#1E5F8A)
Tipografía: Playfair Display (elegancia) + Inter (legibilidad)
📱 Funcionalidades Incluidas
Sistema de Reservas Completo
Selección de experiencia con precios
Calendario de fechas disponibles
Contador de personas
Cálculo automático del total
Formulario de datos del cliente
Simulación de pago (Stripe/PayPal)
Confirmación instantánea
Secciones de la Web
Hero con llamada a la acción clara
Experiencias (3 tours principales)
Tours Privados (CTA especial)
About Us con diferenciadores
Reviews con testimonios verificados
Contacto con formulario + WhatsApp
Footer con enlaces y badges de confianza
🔧 Recomendaciones de Implementación
Opción 1: WordPress + Elementor (Recomendado)
Plugins necesarios:

Amelia o Bookly - Sistema de reservas
WooCommerce + Stripe Gateway - Pagos
WPForms - Formularios de contacto
Yoast SEO - Optimización
WPML - Multiidioma (Español/Inglés)
Opción 2: Desde Celular (Más Simple)
Usa Canva Websites o Carrd.co para empezar rápido, conectando con:

Cal.com (reservas gratis)
PayPal.me (pagos simples)
WhatsApp Business (confirmaciones)
